# docs
安装 docsify-cli
npm i docsify-cli -g
启动本地服务
docsify serve docs

如果想在项目的 ./docs 目录里写文档,直接通过 init 初始化项目。 
docsify init ./docs

启动本地服务
docsify serve docs

# 引言
脱身于admin框架，经过几个月迭代，逐步成为一个通用框架。它极大地简化了开发成本，简化了开发人员的学习成本，并针对部分常用业务提供了针对性的解决方案。
